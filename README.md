#### How To Run
1. Fork or Download The Repo
2. Install All Dependecies

```python
pip install flask
pip install requests
pip install BeautifulSoup4
```

3. Run The Command 

```python
python price_tracker.py
```

4. Your App Will Be Running At `https://127.0.0.1:5000/`
